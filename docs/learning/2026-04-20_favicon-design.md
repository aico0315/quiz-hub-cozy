# ファビコンのデザイン：SVGアイコンの作り方

## 今回作ったもの

ブラウザのタブに表示される小さなアイコン（ファビコン）を、ブルーベリーの星（calyx）モチーフに変更しました。

- **背景**：ダスティモーブ（`#9b7b8f`）の角丸正方形
- **デザイン**：ブルーベリーの最も特徴的な部分（calyx＝先端の星形）を白線で描写
- **ファイル**：`public/favicon.svg`

---

## SVGとは？

SVG（Scalable Vector Graphics）は、数式でグラフィックを描くファイル形式です。
JPGやPNGと違い、どんなサイズに拡大しても画質が劣化しません。ファビコンや各種アイコンに向いています。

---

## 今回使ったSVGの要素

### `<rect>`：四角形

```svg
<rect width="48" height="48" rx="10" fill="#9b7b8f"/>
```

| 属性 | 意味 |
|---|---|
| `width` / `height` | 幅・高さ |
| `rx` | 角丸の半径（大きいほど丸くなる） |
| `fill` | 背景色 |

---

### `<circle>`：円

```svg
<circle cx="33" cy="15" r="3.8" stroke="white" stroke-width="1.8" fill="none"/>
```

| 属性 | 意味 |
|---|---|
| `cx` / `cy` | 円の中心座標 |
| `r` | 半径 |
| `stroke` | 線の色 |
| `stroke-width` | 線の太さ |
| `fill="none"` | 塗りなし（線だけ） |

---

### `<path>`：自由な形

```svg
<path d="M31.5 15 Q31 10.5 33 7.5 Q35 10.5 34.5 15"/>
```

`d` 属性の中にコマンドを書いて自由な形を描きます。

| コマンド | 意味 |
|---|---|
| `M x y` | Move to：その座標に移動（ペンを置く） |
| `Q cx cy x y` | Quadratic bezier：制御点を通る曲線 |
| `Z` | パスを閉じる（始点に戻る） |

---

### `<g>` と `transform="rotate()"` ：グループと回転

5枚の花びらを1つずつ手計算せず、同じ形を回転させて配置しました。

```svg
<!-- 同じ花びら形を72°ずつ回転して5枚配置 -->
<g transform="rotate(0, 33, 15)">
  <path d="M31.5 15 Q31 10.5 33 7.5 Q35 10.5 34.5 15"/>
</g>
<g transform="rotate(72, 33, 15)">
  <path d="...同じpath..."/>
</g>
```

`rotate(角度, 回転の中心X, 回転の中心Y)` の形で書きます。
360 ÷ 5 = 72° ずつ回転することで、5枚が均等に並びます。

## デザインで試したこと

| 試したパターン | 結果 |
|---|---|
| 丸いブルーベリー全体を描く | 爆弾に見えた 😂 |
| 塗りつぶしで描く | ブルーベリーに見えなかった |
| 全体を線だけで描く | 何かわからなかった |
| calyx（星）だけを線で描く | ブルーベリーっぽく見えた ✅ |

**学び：アイコンデザインは「全部描く」より「最も特徴的な部分だけ描く」方が伝わりやすい。**

---

## ブルーベリーの calyx とは？

ブルーベリーの先端にある小さな星形の部分（花の名残）。これがブルーベリーを他の果物と区別する最大の特徴です。5枚の小さな花びら＋中心の丸い輪＋裂け目で構成されています。

---

## ファビコンの設定方法

`public/favicon.svg` に置くだけで、`index.html` の以下の記述が自動で読み込みます。

```html
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
```
